# Libreria.
Es un catálogo de libros interactivo desarrollado como parte del programa (ONE). El objetivo principal de la aplicación es ofrecer a los usuarios una herramienta para buscar libros a través de una API externa, procesar los datos en formato JSON y almacenarlos en una base de datos relacional para su posterior consulta.
🛠️ Funcionalidades Principales
- **Consumo de API**: Obtención de datos reales de libros desde la API de Gutendex.
- **Persistencia en Base de Datos**: Almacenamiento de libros y autores buscados en **PostgreSQL**.
- **Consultas Personalizadas**: 
  - Listar todos los libros guardados.
  - Listar autores registrados.
  - Filtrar autores vivos en un año determinado.
  - Listar libros por idioma.

## ⚙️ Tecnologías y Herramientas
- **Java 17**
- **Spring Boot 3**
- **Spring Data JPA** (Hibernate)
- **PostgreSQL** (Base de Datos)
- **Jackson** (Mapeo de JSON)
- **Maven** (Gestión de dependencias)
